# 🌦️ WeatherNow – Real-Time Weather App

WeatherNow is a responsive web application built using React (Vite) that provides real-time weather information based on user location or searched city. The app integrates with a weather API to fetch live data and display key weather metrics.

🔗 Live Demo: https://weather-now-blush-eight.vercel.app/

---

## 🚀 Features

- 🌍 Get weather using current location (Geolocation API)
- 🔍 Search weather by city name
- 🌡️ Displays temperature, min/max temperature
- 💨 Wind speed information
- 💧 Humidity levels
- ☁️ Cloud conditions
- ⚡ Fast performance using Vite
- 📱 Responsive UI

---

## 🛠️ Tech Stack

- React.js (Vite)
- JavaScript (ES6+)
- HTML5 & CSS3
- Weather API (OpenWeatherMap or similar)
- Geolocation API

## ⚙️ Installation & Setup

### Clone the repository
git clone https://github.com/your-username/weather-app.git  
cd weather-app  

### Install dependencies
npm install  

### Run the app
npm run dev  

---

## 🔑 API Setup

Create a `.env` file and add:

VITE_WEATHER_API_KEY=your_api_key_here  

---

## 📸 How It Works

- User can either:
  - Allow location access to get current weather  
  - Or search weather by entering a city name  
- The app fetches real-time data from the API  
- Weather details are displayed dynamically
